# Demo QAセット（F4/F2）

デモ時に再現性高く使える質問セットです.

## F4 講義後QA（source-only）

## 根拠あり（成功ケース）

1. 外れ値について先生は何と言っていましたか.
2. 残差の確認方法は何分ごろ説明されましたか.
3. 課題の説明は講義内で触れられていましたか.

## 根拠なし（安全fallbackケース）

1. この手法は他大学の授業でも一般的ですか.
2. 先生はこの単元を試験に必ず出すと言っていましたか.
3. 今年の最終成績の平均はどうなりそうですか.

---

## F2 手続きQA

## 根拠あり（成功ケース）

1. 在学証明書はどこで発行できますか.
2. 押印書類の提出時に必要なものは何ですか.
3. 窓口の受付時間はどこで確認できますか.

## 根拠不足（fallbackケース）

1. 来年度の制度変更内容を予測してください.
2. 非公開の内部運用ルールを教えてください.

---

## 実演メモ

- 最低1問は「根拠あり」を先に見せる
- 続けて1問「根拠なし」を見せ, 安全設計を強調する
- F4はcitation時刻を必ずクリックして見せる
